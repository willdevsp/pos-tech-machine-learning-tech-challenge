# models

Esta pasta gerencia o treinamento, a inferência e os artefatos de modelo.

Arquivos principais:
- `baseline.py`: referência de modelo baseline.
- `pipeline.py`: fluxo de treino e validação do modelo.
- `transformers.py`: transformações e codificações de recursos.
- `inference.py`: lógica de geração de previsões com modelo treinado.
- `best_mlp_model.pth`: artefato do modelo salvo.

A pasta reúne tudo que é necessário para construir, testar e servir o modelo de churn.
