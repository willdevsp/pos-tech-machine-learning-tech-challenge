# tests

Esta pasta contém os testes automatizados do projeto.

Arquivos principais:
- `conftest.py`: configura fixtures e configuração comum para os testes.
- `test_config.py`: valida a configuração do projeto.
- `test_models.py`: testa o comportamento dos modelos e pipelines.
- `test_preprocessing.py`: garante o pré-processamento correto dos dados.

Para executar os testes, use `pytest` a partir da raiz do projeto.
